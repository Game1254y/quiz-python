# Quiz sobre Python

Projeto de um quiz de perguntas e respostas feito em Python.

O programa apresenta perguntas com alternativas (A, B, C ou D), embaralha a ordem das perguntas e mostra o resultado final com o número de acertos.

## Funcionalidades

- Perguntas de múltipla escolha
- Perguntas embaralhadas
- Sistema de placar
- Limite de tentativas
- Pausa entre perguntas
- Registro da data e hora do quiz

## Bibliotecas utilizadas

- random → embaralhar perguntas
- time → criar pausas entre perguntas
- datetime → registrar data e hora do quiz

## Como executar

1. Execute o arquivo `Quiz.py`
2. Responda digitando **A, B, C ou D**
3. Veja seu placar ao final do quiz
